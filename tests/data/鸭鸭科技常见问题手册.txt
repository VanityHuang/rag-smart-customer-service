一、产品与服务

问：鸭鸭科技主要提供哪些产品和服务？
答：我们提供三大核心产品线：
    1. 鸭鸭 ERP — 企业资源管理系统，涵盖财务、人事、供应链、库存管理
    2. 鸭鸭 CRM — 客户关系管理平台，支持销售漏斗、客户画像、自动化营销
    3. 鸭鸭 BI  — 商业智能分析工具，提供实时数据看板、自定义报表、AI 预测分析
    此外还提供私有化部署、API 对接、数据迁移等增值服务。

问：产品支持哪些部署方式？
答：支持三种部署方式：
    - SaaS 云端部署：开箱即用，按年订阅，起步价 9800 元/年/10 用户
    - 混合云部署：核心数据本地存储，计算层云端运行
    - 私有化部署：全套部署到客户自有服务器，一次性授权 + 年度维护费

问：是否支持免费试用？
答：所有产品均提供 14 天免费试用，无需绑定信用卡。试用期间可使用全部功能。
    试用到期前 3 天会收到邮件提醒，可选择续费或导出试用数据。
    联系销售可申请延长至 30 天试用期。

问：如何获取技术支持？
答：技术支持渠道如下：
    - 电话热线：400-800-9966（工作日 9:00-18:00）
    - 在线工单：登录后台 → 帮助中心 → 提交工单
    - 企业微信群：购买后由客户经理邀请加入专属服务群
    - 邮件：support@yaya-tech.com
    金牌客户享有 7×24 小时专属技术顾问服务。


二、账户与权限

问：如何创建企业账户？
答：访问 www.yaya-tech.com，点击"免费试用"，填写企业信息完成注册。
    注册后系统自动创建企业空间，注册人默认为管理员。
    管理员可在"组织架构"中邀请成员并分配角色。

问：用户角色有哪些？
答：系统内置以下角色：
    - 超级管理员：拥有所有权限，可管理计费和删除企业
    - 管理员：可管理用户、角色、系统配置，不可操作计费
    - 普通员工：根据分配的功能模块权限操作
    - 外部协作者：仅可访问指定项目，不可查看企业通讯录
    也可自定义角色，精确控制每个功能的查看/编辑/导出权限。

问：忘记密码怎么办？
答：登录页点击"忘记密码"，输入注册邮箱，系统发送重置链接（有效期 30 分钟）。
    若邮箱也无法访问，请联系管理员在后台重置密码。
    企业管理员忘记密码，可通过绑定的手机号接收验证码重置。


三、数据与安全

问：数据存储在哪里？是否安全？
答：SaaS 版本数据存储在阿里云华东2（上海）区域，通过 ISO 27001 和等保三级认证。
    数据加密方案：
    - 传输层：TLS 1.3 加密
    - 存储层：AES-256 加密
    - 密钥管理：使用阿里云 KMS 托管
    支持每日自动备份，备份数据保留 90 天。

问：如何导出我的数据？
答：在"系统设置 → 数据管理"中可导出：
    - 全量数据导出：Excel/CSV 格式，适用于数据迁移
    - 单模块导出：选择具体模块导出
    - API 数据拉取：通过 REST API 程序化获取数据
    导出任务在后台执行，完成后邮件通知下载。

问：是否支持数据删除和账号注销？
答：支持。管理员可在"系统设置 → 账号管理"中发起账号注销。
    注销流程：
    1. 提交注销申请
    2. 7 天冷静期（可随时撤销）
    3. 冷静期过后，所有数据永久删除
    4. 30 天内完成删除并发送确认邮件
    请注意：数据删除后不可恢复，建议提前做好备份。


四、计费与合同

问：如何计费？有哪些套餐？
答：SaaS 版本采用年付制：
    - 基础版：9800 元/年/10 用户，包含核心功能
    - 专业版：29800 元/年/50 用户，含高级分析和自动化
    - 旗舰版：79800 元/年/不限用户，含私有化部署选项
    超出用户数按 580 元/人/年 收费。
    3 年合同享受 8 折优惠，5 年合同享受 7 折优惠。

问：如何开具发票？
答：支持增值税普通发票和增值税专用发票。
    开票流程：后台"财务中心 → 发票管理"填写开票信息。
    电子发票在付款后 3 个工作日内发送到绑定邮箱。
    纸质发票在 7 个工作日内寄出。

问：如何升级或降级套餐？
答：随时可以在"财务中心 → 套餐管理"中操作：
    - 升级：立即生效，按剩余天数补差价
    - 降级：当前计费周期结束后生效，不退差价
    建议在合同到期前 30 天评估是否需要调整。


五、集成与开发

问：支持哪些第三方系统集成？
答：已预置集成的系统包括：
    - 钉钉/企业微信/飞书：单点登录、消息推送
    - 金蝶/用友：财务数据双向同步
    - 顺丰/京东物流：订单物流跟踪
    - 微信支付/支付宝：收款对账
    也提供标准 REST API 和 Webhook，支持与任意系统对接。

问：API 调用有限制吗？
答：API 调用限制根据套餐不同：
    - 基础版：1000 次/天
    - 专业版：10000 次/天
    - 旗舰版：100000 次/天
    超限后返回 HTTP 429 状态码，次日重置。
    如需更高限额，可联系销售购买 API 加油包。
